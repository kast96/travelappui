Мобильное приложение с интерфейсом услуг путешествий

## Getting started

Выполните загрузку зависимостей
```sh
yarn install
```
В зависимости от устройства, на котором будет запущена программа, выполните одну из команд:

**Android**
```sh
yarn android
```
**iOS**
```sh
yarn ios
```
**Web**
```sh
yarn web
```
